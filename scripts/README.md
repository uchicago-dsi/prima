# Shell Helpers

Operational shell helpers live here to keep the repository root focused on Python entrypoints.

- `scripts/sync.sh`: Parallel one-way sync from mounted HIRO share to GPFS.
- `scripts/delete_tarballs.sh`: Quarantine/move tarballs under ChiMEC tree.
- `scripts/fetch_mirai_snapshots.sh`: Download Mirai release snapshots via GitHub CLI.
- `scripts/test_gallery_gen.sh`: Small local gallery generation smoke test.
